# Coin Simulator
Simulates ten trials of three coin flips, displaying the results and the number of heads and tails per trial in a table. Pressing the button generates a new set of randomized results.

## Features
- Simple UI (JavaSwing)
- 10 trials of 3 coin flips (fixed)
- Re-randomized results on button click

## Tech Stack
- Java
- JavaSwing

## How to Run
1. Ensure that Java (JDK) is installed on your system.
2. Save the file
3. Open a terminal or command prompt in the file directory.
4. Compile the program: javac <fileName>.java
5. Run the program: java <fileName>
